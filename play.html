Let's play!

To start please pick a subject first by selecting the icon of your choice, then set the level and click start.

If you need to refresh your memory you can go to the study room. Study

How to play:

You found yourself locked in a labyrinth. You can see the exit but the only way out is to activate some mechanism in order to progress through it. You will need knowledge and logic to make it out.

  • Use the arrows to move through the maze.
  • To activate the mechanism, step on the case and answer the question.
  • If you get it right a passage will open :) .
  • If you get it wrong you will get a strike :( .
  • When the game is over you will be send back to the start for a new game.

The number of strikes before game over depend on the level.
